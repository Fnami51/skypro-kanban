Skypro-kanban

Приложение нужно для организации своих дел в кампании. 
Вы можете зарегестрироваться и создать задачу. При создании задачи вы можете ввести название и описание, выбрать дату и выбрать тему задачи.
Также вы можете изменить задачу. При изменении задачи вы не можете изменить название, но вы можете изменить статус.

Основные задачи:
Регистрация и авторизация
Управление задачами: создание, редактирование, удаление, перемещение по колонкам, работа с календарем
